Pourquoi Nanomus ?
------------------

Plus par passion au départ, puis dans l'idée de créer un outil simple à mettre en place afin de monter un petit site internet, le tout en respectant les standards du W3C, avec des performances exceptionnelles, de manière sécurisé et le tout sans base de données.
De tout ceci Nanomus a vu le jour.
Nanomus est comme son nom l'indique extrêmement minimaliste, un concentré de l'expérience du CMS Maximus et du blog Minimus, qui je l'espère vous permettra de faire ce que vous désirez simplement.

INSTALLATION
-------------
Uploadez le contenu du répertoire
Modifiez si nécessaire, les droits des dossiers et fichiers suivants ( ils doivent être libre en écriture )
  * datas/
  * datas/metas/
  * kernel/config.php

Puis renommez le fichier sample.htaccess en .htaccess

Votre site internet est en place, identifiez vous avec les accès suivants :
 * utilisateur: admin
 * mot de passe: admin


L'installation de Nanomus représente environ 150 ko sur disque ( comprenant images, javascript ,php ... )

ATTENTION toute suppression d'une page est irrémédiable, TOUTES LES DONNEES de cette page seront supprimées, perdues et non remontables par la suite.


Spécificités pour l'hébergeur FREE.FR
-------------------------------------
 Décommentez la ligne 2 du fichier .htaccess
 Modifiez la ligne 37 du fichier index.php suivante
  define('FREE_FR', false);
 comme ceci
  define('FREE_FR', true);
 Nanomus sera ainsi configuré pour free
 Rendez vous ensuite sur votre site, identifiez vous puis IMPORTANT mettez votre URL complète dans la configuration comme ceci: http://monlogin.free.fr/
 NOTE: il faudra IMPERATIVEMENT http:// avant l'url ainsi que le slash final ( / )
